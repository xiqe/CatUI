Popup 弹层

可自定义的弹层组件。

何时使用

需要用户处理事务，又不希望跳转页面以致打断工作流程时，可以使用 Popup 在当前页面正中打开一个浮层，承载相应的操作。

演示 Demo

Parameter

参数 说明 默认值
dom 弹层显示区域 body
mask 是否需要蒙版，可选参数 true false true
width 弹层显示宽度 400
height 弹层显示高度 --（自适应）
paddingTB 弹层纵向间距 10
paddingLR 弹层横向间距 10
bgColor 弹层背景色 #fff
closebtn 是否需要关闭按钮，可选参数 true false true
template 弹层内可以任意添加dom结构 --

Function

方法 说明 参数
.close() 关闭当前弹层 --
