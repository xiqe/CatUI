Top 返回顶部

可自定义的返回顶部组件。

何时使用

当页面过长，导致用户回到页面头部需要长久滚动鼠标滚轴的时候，Top组件可以很好的帮助用户改善这一操作行为。

演示 Demo

Parameter

参数 说明 默认值
duration 滚动速度控制 1000
pageHeight 距离离开顶部相应高度出现返回顶部按钮 300
showSuggest 是否显示意见反馈，可选参数 true false true
showQrcode 是否显示二维码，可选参数 true false true
showPhone 是否显示联系方式，可选参数 true false true
showQQ 是否显示在线客服，可选参数 true false true
