Steps 步骤条

可自定义的步骤条组件。

何时使用

当页面需要分步骤数的时候，Steps将承载分步效果，并且可以随意根据实际需求切换任意步骤。

演示 Demo

Parameter

参数 说明 默认值
container 所要绑定的DOM节点 --
widths 组件总宽度 --
current 初始化当前步骤 --
type 效果可选类型，可选参数 0箭头 1非箭头 0
args 数组类型，输入Steps每一步的Steps title []

Function

方法 说明 参数
.prevStep() 设置跳至上一步 --
.nextStep() 设置跳至下一步 --
.setStep(n) 设置跳转到任意一步 n为number类型
