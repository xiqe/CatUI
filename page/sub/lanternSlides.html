LanternSlides 幻灯片

可自定义的幻灯片组件。

何时使用

需要用户处理事务，又不希望跳转页面以致打断工作流程时，可以使用 Popup 在当前页面正中打开一个浮层，承载相应的操作。

演示 Demo

Parameter

参数 说明 默认值
data 数据来源，{name:'图片Title',imgSrc:'图片URL'} --
width 展示图片宽度 600
height 展示图片高度 400
closeBtn 是否需要关闭按钮，可选参数 true false true
title 是否需要图片title，可选参数 true false true
index 起始图片顺序 1

Function

方法 说明 参数
.close() 关闭当前幻灯片效果 --
.loadImg(num) 跳转到固定的一页 num为页数
